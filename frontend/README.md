# Director Desk Frontend

React + Vite + TailwindCSS scaffold for the Director Desk project.

## Setup

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env` and set the backend URL:

```bash
VITE_API_BASE_URL=http://localhost:8000
```

## Structure

- `src/components/` shared UI pieces for later use
- `src/pages/` route-level screens for future workflow views
- `src/services/` API clients and integration helpers
- `src/hooks/` reusable React hooks
